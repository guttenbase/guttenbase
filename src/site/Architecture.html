GuttenBase architecture

The following image shows the basic architecture of GuttenBase:

Everything is grouped around the connector repository containing all informations about the configured data bases.

  • Data base specific informations are contained in Configurations
  • Hints may influence the migration process in various ways. E.g., by filtering the list of tables to be copied.
  • Tools use the connector repository and the hints to do their work
